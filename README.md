# From Scatter to Line — Linear Regression in Python (Notebook Lab)

A compact Jupyter Notebook project focused on **Linear Regression**: understanding the model, fitting a line to data, and evaluating predictions.

## Goals
- Understand linear regression intuition (relationship between `x` and `y`)
- Fit a model and generate predictions
- Visualize results (scatter + regression line)
- Evaluate performance (e.g., error metrics / basic validation)
